# Gemini Notes Extension

A Gemini CLI extension to help organize and process your notes.

## Features

- **Process Inbox**: Safely moves notes from your `inbox.md` to permanent storage and consolidates todos.
    - Command: `/notes process-inbox`

## Installation

To install this extension locally:

```bash
gemini extensions install .
```

## Usage

Once installed, you can run the processing command from any directory (though it works best where your `inbox.md` lives):

```bash
/notes process-inbox
```
