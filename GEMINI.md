# Gemini Notes Extension

This extension provides tools for processing and organizing notes.

## Capabilities

### Skills

- **Process Inbox** (`process-inbox`): A rigorous workflow for processing an `inbox.md` file. It triages notes, moves them to permanent storage, consolidates tasks, and logs actions, ensuring no data is lost.

## Usage

This extension adds the following command:

- `/notes process-inbox` - Triggers the inbox processing workflow.
